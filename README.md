# multiclip

## 概要


0〜9番のスロットにテキストを保存して、あとから取り出すことができる簡易マルチクリップボードです。  
よく使う定型文やコマンドを登録しておくことで、端末上で素早く再利用できます。

## 動作環境

- Linux または WSL 上のシェル環境
- bash
- Python 3.x

## インストール

```bash
$ git clone https://github.com/kaito958/multiclip-command.git
$ cd multiclip-command
$ bash test.sh
``` 

## 使い方
### スロットに保存する
　　　$ echo "レポート" | ./multiclip 1

標準入力から読んだテキストを、スロット1に保存します。
スロット番号は 0〜9 の好きな番号を指定できます。

### スロットから読み出す
　　　$ ./multiclip --load 1
　　　レポート

スロット1に保存されている内容を標準出力に出力します。
ターミナル上で表示されたテキストを、そのままコピーして再利用できます。

### スロットの一覧を表示する
　　　$ ./multiclip --list
　　　0: empty
　　　1: 〜 bytes
　　　2: empty
　　　...

各スロットが空かどうか、保存されているデータのサイズ（バイト数）を確認できます。

## テスト
　　　$ bash test.sh

すべてのテストが通ると all tests passed と表示されます。
